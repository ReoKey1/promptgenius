🛍️ Store Setup Instructions

Follow these steps to connect your Prompt-U store to Printify and start selling merchandise.

Option 1: Printify + Shopify Integration (Recommended)

  1. Create Printify Account
    • Go to printify.com
    • Sign up for free account
    • No upfront costs - pay only when you make a sale
  2. Upload Designs
    • Use the merchandise logo: /assets/logo-merchandise.png
    • Or use individual t-shirt designs from /assets/store/
    • Recommended: Upload all 3 designs
  3. Create Products
    • Select "Bella+Canvas 3001" (most popular unisex t-shirt)
    • Colors: Black, White, Navy, Heather Gray
    • Sizes: S, M, L, XL, 2XL, 3XL
    • Set retail price: $29-35 (60-70% profit margin)
  4. Connect to Shopify
    • In Printify, go to "My Stores"
    • Click "Add Store" → Select Shopify
    • Follow OAuth connection flow
    • Publish products to Shopify
  5. Update Store Page
    • Replace "Set Up Store" buttons with Shopify product URLs
    • Update store.html with live links

Option 2: Printify + Direct Integration

  1. Get Printify API Key
    • In Printify dashboard, go to Settings → API
    • Generate API key
    • Save securely in environment variables
  2. Use Printify API
    • Documentation: developers.printify.com
    • Create products programmatically
    • Handle orders via webhook
  3. Update Store Buttons
    • Link directly to Printify product pages
    • Or build custom checkout with Stripe

Option 3: Manual Fulfillment

  1. Order Samples
    • Order 1-2 samples of each design
    • Test quality before selling
    • Take professional photos
  2. Set Up Payment
    • Add Stripe or PayPal buttons
    • Collect orders manually
    • Fulfill via Printify as orders come in
